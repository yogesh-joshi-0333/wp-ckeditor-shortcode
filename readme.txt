== Description ==
A lightweight WordPress plugin that enables CKEditor inside textareas using a simple shortcode `[ckeditor]`.  

== Installation ==
1. Download the plugin ZIP file.
2. Go to your WordPress admin panel.
3. Navigate to **Plugins â†’ Add New**.
4. Click **Upload Plugin** and choose the ZIP file.
5. Click **Install Now**, then **Activate** the plugin.

== Usage ==
To insert a CKEditor-enabled textarea, use the following shortcode:

`[ckeditor class="my-class" id="my-editor" height="400px" width="100%" styles="border: 1px solid #ccc; padding: 10px;" text="your text"]`
